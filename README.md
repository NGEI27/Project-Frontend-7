# Project-Frontend-7
A collaborative professional landing page project built with Modular SCSS, HTML5, and CSS . Focused on career growth and responsive design.

🚀 Project-Frontend-7 | CareerPathNow (პირობითი სახელი)

წინამდებარე პროექტი წარმოადგენს მე-7 გუნდის ის ვებ-გვერდის ფრონტენდ ნაწილს. პროექტის მიზანია თანამედროვე, მასშტაბირებადი და ადვილად მართვადი კოდის ბაზის შექმნა.

🛠️ ტექნიკური არქიტექტურა და სტანდარტები
პროექტის საწყის ეტაპზე დაინერგა ინდუსტრიაში მიღებული  პრაქტიკები:

1. Sass 7-1 Architecture
კოდის ორგანიზებისთვის გამოყენებულია Modular SCSS მიდგომა (7-1 Pattern), რაც გულისხმობს სტილების დაყოფას ლოგიკურ მოდულებად:

Abstracts: ცვლადები და ფუნქციები (Variables, Mixins).

Base: საბაზისო სტილები და CSS Reset.

Layout: საიტის ჩონჩხი (Header, Footer, Grid).

Components: დამოუკიდებელი ელემენტები (Buttons, Cards).

ყველა მოდული ცენტრალიზებულად იმპორტირდება main.scss ფაილში.

2. Development Environment (WSL & Git) (ნინო გოგობერიძე)
WSL (Ubuntu): სამუშაო გარემოდ შერჩეულია Linux-ის ქვესისტემა, რაც უზრუნველყოფს გარემოს იდენტურობას საწარმოო (Production) სერვერებთან.

SSH Security: GitHub-თან კომუნიკაცია ხორციელდება უსაფრთხო SSH პროტოკოლით.

Branching Strategy: ვიყენებთ Feature Branching მეთოდოლოგიას. ამჟამად სამუშაოები მიმდინარეობს feature/header-section შტოში, რათა უზრუნველყოფილი იყოს ძირითადი კოდის (Main branch) სტაბილურობა.
